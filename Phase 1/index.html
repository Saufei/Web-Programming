About

Dime is an online financial management which assists you in managing your finances with ease.

Our available features are:

  • Set personal finance goals
  • Manage daily expenses
  • Monthly expenses report with chart
  • Instant import of daily expenses

Set personal finance goals

Allow you to set saving goals and plan to achieve the goals

Manage daily expenses

With this feature you can set your monthly budget/income which
allows you to add, edit, delete every transaction.

Monthly expenses report with chart

Allows you to see the summary of your monthly expenses

Instant import of daily expenses

Import from other sources in the form of Microsoft Excel into your account
